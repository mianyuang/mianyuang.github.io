# React-路由6版本

# 与5版本相比，改变了什么？

  1. 内置组件的变化，移除<Switch/>，新增<Routes />组件

  2. 语法的变化：component={About}变为element={<About />}

  3. 新增多个hook：useParams，useNavigate，useMatch等

  4. 官方明确推荐函数式组件了

# Component的使用

<Navigate>

  1. 作用：只要<Navigate>组件被渲染，就会修改路径，切换视图

  2. replace属性用于控制跳转模式（push或replace，默认是push）

  3. 示例代码：可以做重定向路由

    import React from 'react'
import {NavLink,Route,Routes,Navigate} from 'react-router-dom'
import Home from './Home'
import About from './About'
export default function Demo() {
  return (
    <div>
        <div>
            <NavLink to='/about'>About</NavLink>
            <NavLink to='/home'>Home</NavLink>
        </div>
        <div>
            <Routes>
                <Route path='/about' element={<About></About>} />
                <Route path='/home' element={<Home></Home>} />
                <Route path='/' element={<Navigate to='/about' ></Navigate>}></Route>
            </Routes>
        </div>
    </div>
  )
}

# <Routes/> 与<Route />

  1. v6版本移除了Switch组件，引进了新的代替者<Routes>

  2. <Routes>与<Route>要配合使用，且必须要用<Routes>包裹<Route>

  3. <Route>相当于一个if语句，如果其路径与当前路径URL匹配，则呈现其对应的组件

  4. <Route caseSensitive>属性用于指定：匹配时是否区分大小写（默认false）

  5. 当URL发生变化时，<Routes>都会查看其所有子<Route>元素以找到最佳匹配并呈现组件

  6. <Route>也可以嵌套使用，且可以配合useRoutes()配置路由表，但需要通过<outlet>组件来渲染其子路由

<Routes>
    <Route path='/about' element={<About></About>} />
    <Route path='/home' element={<Home></Home>} />
    <Route path='/' element={<Navigate to='/about' ></Navigate>}></Route>
</Routes>

# NavLink

自定义高亮样式可以通过className属性赋值一个函数指定或者style接收一个函数指定。

该函数接收一个{active}参数表示是否激活来返回你想要的自定义样式。

属性end表示在当前路由下在点击它的子路由，会把当前路由高亮效果去掉

import React from 'react'
import {NavLink,Route,Routes,Navigate} from 'react-router-dom'
import Home from './Home'
import About from './About'
export default function Demo() {
  return (
    <div>
        <div>
            <NavLink to='/about' style={({isActive})=> isActive?{color:'yellow'}:{color:'blue'}}>About</NavLink>
            <NavLink to='/home' style={({isActive})=>isActive?{color:'yellow'}:{color:'blue'}}>Home</NavLink>
        </div>
        <div>
            <Routes>
                <Route path='/about' element={<About></About>} />
                <Route path='/home' element={<Home></Home>} />
                <Route path='/' element={<Navigate to='/about' ></Navigate>}></Route>
            </Routes>
        </div>
    </div>
  )
}

# 嵌套路由使用

  1. 编写好路由,主要是chilrren属性

     const routes = useRoutes([
    { path: "/about", element: <About /> },
    { path: "/home", element: <Home />, children: [{path:"news",element:<New></New>},{path:'message',element:<Message></Message>}] },
    { path: "/", element: <Navigate to="/about" /> },
  ]);

  2. 使用OutLet组件占用位置，类似vue中的router-view，把路由组件显示在该位置

    import React from 'react'
import { NavLink,Outlet } from 'react-router-dom'

export default function Home() {
  return (
    <div>
        <div>
            <NavLink to='news'>news</NavLink>
            <NavLink to='message'>message</NavLink>
        </div>
        <div>
            <Outlet></Outlet>
        </div>
    </div>
  )
}

# 路由传参

  1. 传递params参数

    1. NavLink组件写好要传递的params参数

      <NavLink to={`detail/${m.id}/${m.title}/${m.content}`}>{m.title}</NavLink>

    2. useRoutes写好路由接收参数数

      {
   path:'detail/:id/:title/:content',
   element:<Detail></Detail>
}

    3. useParams使用接收过来的参数

      const {id,title,content} = useParams()

  2. 传递search参数

    1. NavLink组件写好要传递的search参数

      <NavLink to={`detail?id=${m.id}&title=${m.title}&content=${m.content}`}>{m.title}</NavLink>

    2. useRoutes写好路由接收参数数

      {
   path:'detail',
   element:<Detail></Detail>
}

    3. useSearchParams使用接收过来的参数

       const [search] = useSearchParams()
  let id,title,content ;
  id = search.get('id');
  title = search.get('title');
  content = search.get('content');

  3. 传递state参数

    1. NavLink组件写好要传递的state参数

      <NavLink to='detail' state={{id:m.id,title:m.title,content:m.content}}>{m.title}</NavLink>

    2. useRoutes写好路由接收参数数

      {
   path:'detail',
   element:<Detail></Detail>
}

    3. useLocation使用接收过来的参数

      const {id,title,content} = useLocation().state

# React应用（基于React脚手架）

# 创建项目并启动

全局安装：npm i -g create-react-app

使用命令创建项目：create-react-app 项目名

启动项目：npm run

# 项目结构

解析项目入口文件index.js核心内容：

import React from 'react';//引入react核心库
import ReactDOM from 'react-dom/client';//引入reactDOM库
import App from './App';//引入App组件，一般所有的组件都写到App里面去
//获取index.html页面的root节点，把内容都渲染到该节点上
const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
);

解析App组件：

import React,{Component} from "react";
import Hello from './components/Hello'
//创建并暴露APP组件，它作为外层的一个盒子容器，里面放入所需要渲染的组件
export default class App extends Component{
    render(){
        return (
           <Hello />
        )
    }
}

# 注意点：

  1. 拆分组件、实现静态文件，注意className、style写法。

  2. 多组件初始化状态，如何确定将数据放在哪个组件的state中？

    1. 某个组件使用：放在自身的state中

    2. 多个组件共同使用：放在他们共同的父组件state中，称为状态提升。

  3. 父子组件通信：

    1. 父组件→子组件：通过props传递数据

    2. 子组件→父组件：通过props传递数据，要求父给子传递一个函数props属性。

  4. 状态在哪里，操作状态的方法就在哪里

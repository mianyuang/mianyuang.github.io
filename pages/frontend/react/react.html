# react

一个构建用户界面的js库，更方便操作DOM元素。

# 为什么使用react？

如果使用原生js操作htmlDOM节点交互，交互繁琐，效率低下。采用react，使用虚拟DOM技术，效率高。

# 数据在渲染到页面时：

  • 不使用虚拟DOM情况：

    异步加载获取数据，把数据渲染直接渲染到页面，当再一次获取数据，如果数据前几项是相同的，也会重新渲染。

  • 使用虚拟DOM情况：

    异步加载获取数据，先把数据转换成虚拟DOM，在渲染到页面，当再一次获取数据，转换成虚拟DOM，会先比较上次虚拟DOM中是否有相同的数据，有直接拿过来使用，不会重新渲染。

# 初体验

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>

</head>

<body>
    <!-- 定义空的容器 -->
    <div id="test"></div>
    <!-- 加载 React。-->
    <!-- 注意: 部署时，将 "development.js" 替换为 "production.min.js"。-->
    <script src="./react.development.js" type="text/javascript" ></script>
    <script src="./react-dom.development.js" type="text/javascript"></script>
    <script src="./babel.min.js" type="text/javascript"></script>
    <script type="text/babel">
        //1、定义虚拟DOM 使用JSX模式
        const VDOM = <h1>hello React</h1>
        //2、渲染
        ReactDOM.render(VDOM, document.getElementById('test'))
    </script>
</body>

</html>

# 关于虚拟DOM

  • 本质就是一个Object类型的对象

  • 虚拟DOM比较轻，属性比较少，而真实DOM属性最全。

  • 虚拟DOM最终会被React转化为真实DOM，呈现在页面上。
